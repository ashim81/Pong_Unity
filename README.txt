Pong Game – Unity (CS3383 Assignment)

Author: Asim Sapkota

---------------------------------------
OVERVIEW
---------------------------------------
This project is a simple Pong game developed using Unity and C#.
The player controls a paddle to hit a moving ball while an AI opponent
plays against the user. The game includes a help screen that explains
the controls and pauses the game when opened.

---------------------------------------
CONTROLS
---------------------------------------
Up / Down Arrow Keys : Move player paddle
Help Button                  : Open help screen (pauses game)
ESC                          : Quit the game

---------------------------------------
RUNNING THE GAME
---------------------------------------
To run the compiled game:
1. Download and unzip the submission folder.
2. Double-click pong.exe to start the game.
3. Keep all generated folders in the same directory as pong.exe.

---------------------------------------
OPENING AND BUILDING IN UNITY
---------------------------------------
1. Download the project as a ZIP file.
2. Extract the files to a folder on your computer.
3. Install Unity Hub from https://unity.com/download.
4. Open Unity Hub and select “Open Project”.
5. Choose the extracted project folder.
6. Open Assets/Scenes/SampleScene.
7. Click the Play button to run the game.
8. To build the game, go to File → Build Profiles and select Build.

---------------------------------------
HELP SCREEN
---------------------------------------
The help screen can be opened at any time using the Help button.
While the help screen is visible, the game is paused.

---------------------------------------
NOTES
---------------------------------------
This is an individual assignment developed on macOS.
The Windows executable (pong.exe) is included as required.
